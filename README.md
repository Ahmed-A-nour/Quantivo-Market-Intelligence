# 🌐 Quantivo | Strategic Market Intelligence Dashboard

## 🚀 Project Overview
A high-end Business Intelligence tool developed to provide executive-level insights into North American market performance. This dashboard visualizes sales trends, regional profitability, and portfolio distribution using a modern dark-themed interface.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **Visualization:** Plotly Express (Mapbox, Sunburst, Area Charts)
- **Data Handling:** Pandas, NumPy

## ✨ Key Features
- **Interactive Geospatial Mapping:** Real-time visualization of sales and profit density across Canada and the US.
- **Hierarchical Portfolio Analysis:** Multi-level breakdown of categories (Data Analytics, Tech Support, etc.).
- **Executive Metrics (KPIs):** Instant access to Gross Revenue, Net Profit, and Operating Margins.
